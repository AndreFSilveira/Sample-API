Books API

Creating an book

POST /v1/books

First, create an book, then make a later request to get it back

Parameters

Name Description
book[name] Name of book
book[pages] Pages of book
book[leased] If book is leased
book[year] year of book launch
book[customer_id] Customer who rented the book

Response Fields

Name Description
book[name] Name of book
book[pages] Pages of book
book[leased] If book is leased
book[year] year of book launch
book[customer_id] Customer who rented the book
